# Task Tracker MVP

## Опис проєкту
Web-додаток для управління завданнями. [cite_start]Розроблено в рамках виконання практичної роботи №6 з дисципліни "Web-технології та Web-дизайн". [cite_start]Додаток являє собою MVP (Minimum Viable Product) версію таск-трекера.

## Реалізований функціонал (Скоуп MVP)
* Додавання нових завдань через текстове поле.
* Відображення списку всіх завдань.
* Зміна статусу завдання (виконано/не виконано).
* Видалення завдань зі списку.
* Локальне збереження даних (LocalStorage) для відновлення стану після перезавантаження сторінки.

## Технологічний стек
* Бібліотека: React
* Збирач: Vite
* Стилізація: Tailwind CSS
* Мова: JavaScript
