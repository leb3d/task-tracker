# Task Tracker MVP

## Опис проєкту
Web-додаток для управління завданнями. [cite_start]Розроблено в рамках виконання практичної роботи №6 з дисципліни "Web-технології та Web-дизайн"[cite: 1]. [cite_start]Додаток являє собою MVP (Minimum Viable Product) версію таск-трекера[cite: 1].

## Реалізований функціонал (Скоуп MVP)
* Додавання нових завдань через текстове поле.
* Відображення списку всіх завдань.
* Зміна статусу завдання (виконано/не виконано).
* Видалення завдань зі списку.
* Локальне збереження даних (LocalStorage) для відновлення стану після перезавантаження сторінки.

## Технологічний стек
* Бібліотека: React
* Збирач: Vite
* Стилізація: Tailwind CSS
* Мова: JavaScript

## Інструкція із запуску (Локальне середовище)

1. Клонувати репозиторій:
```bash
git clone [https://github.com/leb3d/task-tracker.git](https://github.com/leb3d/task-tracker.git)

```

2. Перейти до директорії проєкту:

```bash
cd task-tracker

```

3. Встановити залежності:

```bash
npm install

```

4. Запустити локальний сервер розробки:

```bash
npm run dev

```
